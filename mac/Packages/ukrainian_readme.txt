Norman розкладки клавіатури упаковується як keylayout пачці, в тому числі і інший значок схожий за стилем на системні іконки для інших вбудованих макетів, як Dvorak або Colemak.

Норман буде додано як опція джерела вхідного сигналу для мови і тексту в системних настройках.

Норман заснований на американській розкладки клавіатури з тими ж мертвими ключі переміщені в різних місцях.

Щоб видалити розташування, зніміть Нормана пачка з / Library / Клавіатура Розкладка /.

Ukelele був використаний для створення макета:
http://scripts.sil.org/ukelele

iConvert значки, щоб зробити значок меню NO:
http://iconverticons.com/
