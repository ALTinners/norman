layout di tastiera Norman è confezionato come un fascio keylayout, tra cui un'altra icona simile nello stile a icone di sistema per altri layout built-in come Dvorak o Colemak.

Norman verrà aggiunto come opzione della sorgente di ingresso per Lingua e testo in Preferenze di Sistema.

Norman si basa sul layout di tastiera degli Stati Uniti con gli stessi tasti morti spostati in posizioni diverse.

Per disinstallare il layout, rimuovere il Norman fascio da / Library / tastiera I layout /.

Ukulele è stato utilizzato per creare il layout:
http://scripts.sil.org/ukelele

iConvert icone per rendere l'icona del menu NO:
http://iconverticons.com/
