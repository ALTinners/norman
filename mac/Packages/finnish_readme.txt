Norman näppäimistöasettelu on pakattu keylayout nippu, mukaan lukien vaihtoehtoinen kuvake samanlainen tyyli järjestelmään kuvakkeet muita sisäänrakennettu kaavoista kuten Dvorak tai Colemak.

Norman lisätään tulona Source vaihtoehto Kieli ja teksti Järjestelmäasetuksissa.

Norman perustuu Yhdysvaltain näppäimistöasettelu samalla kuollut avaimet siirtää eri paikkoihin.

Jos haluat poistaa asettelun, poista Norman nippu / Library / näppäimistöasettelut /.
