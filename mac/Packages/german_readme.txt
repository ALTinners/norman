Das normannische Tastaturlayout ist als Keylayout-Bundle verpackt, einschließlich eines alternativen Symbols ähnlich dem Stil der System-Icons für andere integrierte Layouts wie Dvorak oder Colemak.

Norman wird als Eingabequelle für Sprache & Text in den Systemeinstellungen hinzugefügt.

Norman basiert auf dem US-Tastaturlayout, wobei die gleichen toten Tasten an verschiedene Orte verschoben wurden.

Um das Layout zu deinstallieren, entfernen Sie das Norman-Bundle aus / Library / Keyboard Layouts /.

Ukelele wurde verwendet, um das Layout zu erstellen:
http://scripts.sil.org/ukelele
