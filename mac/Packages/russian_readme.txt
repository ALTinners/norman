Norman раскладки клавиатуры упаковывается как keylayout пачке, в том числе и другой значок похож по стилю на системные иконки для других встроенных макетов, как Dvorak или Colemak.

Норман будет добавлен в качестве опции источника входного сигнала для языка и текста в системных настройках.

Норман основан на американской раскладки клавиатуры с теми же мертвыми ключи перемещены в разных местах.

Чтобы удалить расположение, снимите Нормана пачка из / Library / Клавиатура Раскладка /.

Ukelele был использован для создания макета:
http://scripts.sil.org/ukelele

iConvert значки, чтобы сделать значок меню NO:
http://iconverticons.com/
